# 日本株ウォッチ 📈

毎日の日本株を、初心者にもやさしくチェックできるローカルWebアプリです。

- 🔄 **最新の株価**を毎日チェック（Yahoo Financeから自動取得）
- ⭐ **短期・中期・長期**で注目候補をスコア表示
- 📖 「**なぜ注目候補なのか**」を初心者向けに解説
- 📰 **日本株に影響しそうな最新ニュース**をテーマ別に表示（ファンダメンタル）＋「なぜ重要か」解説
- 💼 買った株を**登録**しておくと、**売り時のサイン**を教えてくれる
- 🧠 「**なぜ売るべきか**」もやさしく解説
- 🔍 銘柄検索・ウォッチリスト管理・用語集つき

> ⚠️ **重要なお願い**
> 本アプリはテクニカル指標にもとづく**情報提供・学習**のためのツールです。
> 表示される「注目候補」「売却の目安」は将来を保証するものではなく、
> **投資勧誘・投資助言ではありません**。最終的な投資判断はご自身の責任で行ってください。

## 起動のしかた

### かんたん（おすすめ）
1. `起動.command` を**ダブルクリック**します。
2. 黒い画面（ターミナル）が開き、自動でブラウザにアプリが表示されます。
3. 使い終わったら、ターミナルの画面で `Ctrl + C` を押すと終了します。

> 🔒 初回だけ「開発元を確認できないため開けません」と出ることがあります。
> その場合は `起動.command` を **右クリック →「開く」→「開く」** を選べば起動できます。

### ターミナルから
```bash
cd "/Users/niijimayutaka/Desktop/株分析/app"
python3 app.py
```
起動後、ブラウザで http://127.0.0.1:8765/ を開きます（自動で開きます）。

### 📱 iPhoneで使いたい
- **同じWi-Fi内＋Macが起動している**なら：`iPhone起動.command` をダブルクリックし、ターミナルへ表示される `http://192.168.x.x:8765/` をiPhoneのSafariに入力します。
- **Macを閉じていても・外出先でも使いたい**なら：無料クラウドに公開します。手順は **[DEPLOY.md](DEPLOY.md)** を参照（公開URLはPINで保護できます）。

## 使い方

| タブ | できること |
|------|-----------|
| **注目候補** | 短期/中期/長期を選ぶと、勢いやトレンドの良い順に注目候補が並びます。各銘柄の理由をタップすると解説が開きます。「くわしく見る」で詳細分析＋関連ニュース。 |
| **ニュース** | 日本株に影響しそうな最新ニュースをテーマ別（主要・決算・金利為替・半導体・経済）に表示。「💡なぜ重要？」で値動きとの関係を解説します。記事タイトルをクリックすると元記事が開きます。 |
| **マイ株（保有）** | 買った株を登録。毎日の株価から「保有継続／売却検討／損切り検討」を判定し、理由を解説します。 |
| **銘柄を探す** | 会社名・コード・業種で検索。ウォッチリスト（分析対象）への追加もここから。 |
| **使い方・用語** | 株の用語をやさしく解説。RSI、移動平均線、ゴールデンクロスなど。 |

右上の **「更新」** ボタンで、最新の株価を取り直して再分析します。毎朝・毎晩開いて押すのがおすすめです。

## 仕組み（かんたんに）

- 株価は [Yahoo Finance](https://finance.yahoo.com/) の公開データ、ニュースは [Google ニュース](https://news.google.com/) のRSSを使っています（どちらもAPIキー不要・無料）。
- 株価は約2年分を取得し、200日移動平均や長期トレンドも判定しやすくしています。
- PER/PBR/配当利回り/ROEなどの財務指標は、取得できる場合だけ長期評価に加味します。取得元の制限で取れない場合は、画面に「未取得」と表示し、テクニカル指標だけで分析します。
- 取得したデータは `app/data/cache/` に一時保存し、株価は約30分・財務指標は約12時間・ニュースは約15分は再利用します（表示を速く・サーバに優しく）。
- 登録した**保有銘柄・ウォッチリストは、見ているブラウザ（iPhone/PC）の中**に保存されます（localStorage）。サーバーには残らないので、クラウドに公開しても他人に見られず、サーバー再起動でも消えません。
- `APP_PIN` を設定すると、アプリを開く前にPIN入力が必要になります。クラウド公開時は必ず設定してください。

### おすすめ・売り時の判断に使っている指標
移動平均線（5/25/75/200日）、RSI、MACD、ボリンジャーバンド、ATR、出来高、52週高値・安値、ゴールデン／デッドクロス、PER/PBR/配当利回り/ROE（取得できた場合）など。
それぞれの意味はアプリ内の「用語集」で解説しています。

## 困ったとき

| 症状 | 対処 |
|------|------|
| `python3` が見つからない | ターミナルで `python3 --version` を実行。出なければ [python.org](https://www.python.org/) からインストール。 |
| ポートが使用中（Address already in use） | 別のポートで起動：`PORT=8000 python3 app.py` |
| データが「取得失敗」になる | ネット接続を確認し、少し待って「更新」を押す（一時的なアクセス制限の場合があります）。 |
| 株価が古い | 右上「更新」を押すと最新化します。 |

## カスタマイズ

- **分析する銘柄を増やす**：アプリの「銘柄を探す」から検索してウォッチリストに追加。
- **初期銘柄リストを編集**：`app/stocks.py` の `MASTER` / `DEFAULT_WATCHLIST` を編集。
- **損切りラインの初期値**：保有登録時に銘柄ごとに設定できます（未指定は -8%）。
- **キャッシュ時間**：`app/data.py` の `CACHE_TTL`（秒）。
